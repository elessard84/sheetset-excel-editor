Privacy Policy

Sheet Set Excel Editor

Sheet Set Excel Editor does not collect, transmit, store, sell, or share personal information.

All processing is performed locally on the user's workstation.

No user data is sent to external servers.

No analytics, tracking, telemetry, or advertising services are used.

The application works with local AutoCAD Sheet Set DST files and Excel XLSX files selected by the user.
