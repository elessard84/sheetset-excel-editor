Sheet Set Excel Editor

Export, edit and import AutoCAD Sheet Set data using Excel.

Overview

Sheet Set Excel Editor allows AutoCAD and Civil 3D users to export Sheet Set data to Excel, make bulk edits, validate changes, and safely import updates back into the Sheet Set.

Features

Export

Export Sheet Set data to XLSX format.

Edit

Modify custom properties using Excel.

Validate

Review changes before updating the DST file.

Import

Apply validated changes back to the Sheet Set.

Workflow

DST → Export XLSX → Edit in Excel → Validate → Import → DST

Compatibility

  • AutoCAD 2026
  • Civil 3D 2026
